# AI Hiring Intelligence Agent - Frontend

This is the frontend for the AI Hiring Intelligence Agent, designed to be deployed on Hugging Face Spaces (static) and connect to a backend on Vercel.

## Setup
1. `npm install`
2. `npm run dev`

## Deployment
- **Frontend**: Upload to Hugging Face Space (Static SDK).
- **Backend**: Deploy to Vercel. Use the `VITE_API_URL` environment variable to point researchers to your backend.
